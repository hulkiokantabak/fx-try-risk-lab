      Method

      What this lean browser edition does

      This site is a static GitHub Pages app. It publishes a prebuilt daily snapshot using public ECB, FRED, Cboe, CBRT, and RSS data, then scores TRY depreciation risk from five lenses: price action, volatility, global macro, Turkey policy/reserves, and headline intensity.

      The goal is to lead with one clear read first, then show the evidence below it. It is not a trading instruction engine and it does not promise perfect forecasts.
